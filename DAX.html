DAX Functions – Complete Explanation Guide

1. Aggregation Functions

SUM()

Explanation: SUM() is used to add all numeric values in a column. It is commonly used to calculate totals such as total sales, total marks, or total revenue. SUM(Table[Column]) Example: TotalSales = SUM(Sales[Amount])

AVERAGE()

Explanation: AVERAGE() calculates the mean of numeric values in a column. AVERAGE(Table[Column]) Example: AverageSalary = AVERAGE(Employee[Salary])

COUNT()

Explanation: COUNT() counts only numeric values in a column. COUNT(Table[Column]) Example: TotalOrders = COUNT(Orders[OrderID])

COUNTA()

Explanation: COUNTA() counts all non-blank values including text. COUNTA(Table[Column]) Example: TotalStudents = COUNTA(Student[Name])

COUNTBLANK()

Explanation: COUNTBLANK() counts empty or null values in a column. COUNTBLANK(Table[Column]) Example: MissingEmails = COUNTBLANK(Student[Email])

2. Logical Functions

IF()

Explanation: IF() checks a condition and returns different results for TRUE and FALSE. IF(condition, value_if_true, value_if_false) Example: Result = IF(Marks[Score] >= 40, "Pass", "Fail")

AND()

Explanation: AND() returns TRUE only if all conditions are true. AND(condition1, condition2) Example: Eligible = IF(AND(Age >= 18, Age <= 60), "Yes", "No")

OR()

Explanation: OR() returns TRUE if any one condition is true. OR(condition1, condition2) Example: Discount = IF(OR(Sales > 50000, CustomerType = "VIP"), "Yes", "No")

SWITCH()

Explanation: SWITCH() replaces multiple IF conditions and improves readability. SWITCH(expression, value1, result1, else) Example: Grade = SWITCH(TRUE(), Marks >= 75, "A", Marks >= 60, "B", Marks >= 40, "C", "Fail")

3. Date and Time Functions

TODAY()

Explanation: Returns the current system date. TODAY() Example: CurrentDate = TODAY()

YEAR()

Explanation: Extracts the year from a date. YEAR(date) Example: OrderYear = YEAR(Orders[OrderDate])

DATEDIFF()

Explanation: Calculates difference between two dates in days, months, or years. DATEDIFF(start_date, end_date, unit) Example: Experience = DATEDIFF(Employee[JoinDate], TODAY(), YEAR)

4. Text Functions

CONCATENATE()

Explanation: Joins two text values into one. CONCATENATE(text1, text2) Example: FullName = CONCATENATE(Student[FirstName], Student[LastName])

LEFT()

Explanation: Extracts characters from the left side of text. LEFT(text, num_chars) Example: Code = LEFT(Product[ProductCode], 3)

LEN()

Explanation: Returns number of characters in a text. LEN(text) Example: NameLength = LEN(Customer[Name])

5. Filter Functions

FILTER()

Explanation: Returns a filtered table based on condition. FILTER(table, condition) Example: HighSales = FILTER(Sales, Sales[Amount] > 10000)

ALL()

Explanation: Removes all filters from a table or column. ALL(table/column) Example: TotalSalesAll = CALCULATE(SUM(Sales[Amount]), ALL(Sales))

6. Relationship Functions

RELATED()

Explanation: Fetches value from a related table using relationship. RELATED(column) Example: DepartmentName = RELATED(Department[DeptName])

7. Table Functions

SUMMARIZE()

Explanation: Groups data and creates summary tables. SUMMARIZE(table, groupBy_column, "Name", expression) Example: SalesSummary = SUMMARIZE(Sales, Sales[Region], "TotalSales", SUM(Sales[Amount]))

8. Mathematical Functions

DIVIDE()

Explanation: Safely divides two numbers and avoids divide-by-zero error. DIVIDE(numerator, denominator, alternateResult) Example: ProfitMargin = DIVIDE(Profit, Sales, 0)
