Power BI – Data Transformation & Cleaning Notes (Power Query)

1. What is Data Transformation in Power BI?

Data Transformation means converting raw, unclean data into a structured and analysis-ready format.

In Power BI, this is done using Power Query Editor.

Why it is needed?

  • Raw data may contain null values
  • Duplicate records
  • Wrong data types
  • Extra spaces
  • Inconsistent formats

Power Query helps clean and shape data before loading it into the data model.

2. Power Query Editor – How to Open

Steps:

  1. Open Power BI Desktop
  2. Click Home → Get Data
  3. Select data source (Excel / CSV / SQL / etc.)
  4. Click Transform Data

This opens Power Query Editor.

3. Understanding NULL Values

What is a NULL value?

  • A NULL means missing or unknown data
  • It is not zero (0) and not empty string ("")

Examples:

  • Student marks not entered
  • Salary not disclosed
  • Date not available

4. How to Identify NULL Values

Steps:

  1. Open Power Query Editor
  2. Select a column
  3. Look for cells showing null
  4. Filter icon → you will see (null) option

5. Remove Rows Containing NULL Values

Case 1: Remove rows with NULL in specific column

  1. Select the column
  2. Click Home → Remove Rows → Remove Blank Rows

OR

  1. Click filter icon of column
  2. Uncheck (null)
  3. Click OK

When to use?

  • When NULL values are meaningless
  • When incomplete records must be discarded

6. Replace NULL Values

Example:

Replace NULL marks with 0

Steps:

  1. Select the column
  2. Go to Transform → Replace Values
  3. Value to find: null
  4. Replace with: 0
  5. Click OK

Use cases:

  • Numerical calculations
  • Avoid errors in aggregation

7. Fill NULL Values (Fill Down / Fill Up)

Used when data is grouped and NULLs depend on previous values.

Fill Down – Steps:

  1. Select column
  2. Transform → Fill → Down

Fill Up – Steps:

  1. Select column
  2. Transform → Fill → Up

Example: Department name missing in rows below first occurrence

8. Add NULL Value Column

Create a new column with NULL values

  1. Go to Add Column → Custom Column
  2. Column name: Remarks
  3. Formula: null
  4. Click OK

Used when:

  • Placeholder columns are needed
  • Future data entry

9. Handling Duplicate Values

What are duplicates?

Multiple rows having identical data

Identify duplicates – Steps:

  1. Select column(s)
  2. Home → Keep Rows → Keep Duplicates

10. Remove Duplicate Rows

Remove duplicates based on one column

  1. Select column
  2. Home → Remove Rows → Remove Duplicates

Remove duplicates based on multiple columns

  1. Select multiple columns (Ctrl + Click)
  2. Home → Remove Rows → Remove Duplicates

11. Change Data Types

Why important?

  • Calculation errors
  • Incorrect visuals

Steps:

  1. Select column
  2. Transform → Data Type
  3. Choose correct type (Text, Whole Number, Decimal, Date, etc.)

12. Trim, Clean & Format Text Data

Trim (remove extra spaces)

  1. Select text column
  2. Transform → Format → Trim

Clean (remove non-printable characters)

Transform → Format → Clean

Change case

  • Uppercase
  • Lowercase
  • Capitalize Each Word

13. Split Column

Example:

Full Name → First Name & Last Name

Steps:

  1. Select column
  2. Transform → Split Column → By Delimiter
  3. Choose delimiter (space, comma, etc.)

14. Merge Columns

Combine multiple columns

  1. Select columns
  2. Transform → Merge Columns
  3. Choose separator

15. Conditional Column

Example:

If Marks ≥ 40 → Pass else Fail

Steps:

  1. Add Column → Conditional Column
  2. Set conditions
  3. Name the column

16. Filter Rows

Remove unwanted data

  1. Click filter icon
  2. Apply conditions (equals, greater than, contains)

17. Sort Data

  1. Click column header
  2. Choose Ascending / Descending

18. Remove Errors

Used when calculation or type conversion fails

Steps:

  1. Select column
  2. Home → Remove Rows → Remove Errors

19. Rename Columns

  1. Double-click column header
  2. Enter meaningful name

20. Close & Apply

Steps:

  1. Click Home → Close & Apply
  2. Data loads into Power BI model
