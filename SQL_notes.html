1. What is SQL?

SQL (Structured Query Language) is a standard language used to store, retrieve, modify and manage data in relational databases. It is widely used in banking, universities, data science, cyber security and enterprise systems.

2. CREATE TABLE

CREATE TABLE students ( roll_no INT PRIMARY KEY, name VARCHAR(50), age INT, marks FLOAT );

3. INSERT Data

INSERT INTO students VALUES (1, 'Rahul', 20, 85.5); INSERT INTO students VALUES (2, 'Amit', 21, 78.0);

4. SELECT Data

SELECT * FROM students; SELECT name, marks FROM students;

5. WHERE Clause

SELECT * FROM students WHERE marks > 80;

6. UPDATE

UPDATE students SET marks = 90 WHERE roll_no = 1;

7. DELETE

DELETE FROM students WHERE roll_no = 2;

8. Aggregate Functions

SELECT COUNT(*) FROM students; SELECT AVG(marks) FROM students; SELECT MAX(marks) FROM students;

9. GROUP BY & HAVING

SELECT age, AVG(marks) FROM students GROUP BY age HAVING AVG(marks) > 80;

10. JOIN

SELECT students.name, marks.score FROM students INNER JOIN marks ON students.roll_no = marks.roll_no;

11. TRUNCATE & DROP

TRUNCATE TABLE students; DROP TABLE students;
