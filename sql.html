Introduction to SQL

SQL (Structured Query Language) is a standard language used to interact with relational databases. It allows you to create, read, update, and delete data. 

-- Example: Select all records from a table
SELECT * FROM employees;

DDL (Data Definition Language)

DDL commands are used to define or modify database structures such as tables.

1. CREATE

Used to create databases and tables.

-- Syntax:
CREATE TABLE table_name (
    column1 datatype PRIMARY KEY,
    column2 datatype,
    column3 datatype
);

-- Example:
CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    salary DECIMAL(10,2)
);

2. ALTER

Used to modify an existing table.

-- Syntax:
ALTER TABLE table_name
ADD column_name datatype;

-- Example:
ALTER TABLE employees
ADD department VARCHAR(50);

3. DROP

Used to delete tables or databases.

-- Syntax:
DROP TABLE table_name;

-- Example:
DROP TABLE employees;

DML (Data Manipulation Language)

DML commands are used to manipulate the data stored in tables.

1. INSERT

-- Syntax:
INSERT INTO table_name (column1, column2)
VALUES (value1, value2);

-- Example:
INSERT INTO employees (id, name, salary)
VALUES (1, 'John Doe', 50000);

2. UPDATE

-- Syntax:
UPDATE table_name
SET column1 = value1
WHERE condition;

-- Example:
UPDATE employees
SET salary = 55000
WHERE id = 1;

3. DELETE

-- Syntax:
DELETE FROM table_name
WHERE condition;

-- Example:
DELETE FROM employees
WHERE id = 1;

DCL (Data Control Language)

DCL commands control access to data.

1. GRANT

-- Syntax:
GRANT privilege ON object TO user;

-- Example:
GRANT SELECT, INSERT ON employees TO 'user1';

2. REVOKE

-- Syntax:
REVOKE privilege ON object FROM user;

-- Example:
REVOKE INSERT ON employees FROM 'user1';

TCL (Transaction Control Language)

TCL commands manage transactions in the database.

1. COMMIT

-- Syntax:
COMMIT;

-- Example:
-- After inserting or updating records, save changes
INSERT INTO employees (id, name, salary) VALUES (2, 'Jane Doe', 60000);
COMMIT;

2. ROLLBACK

-- Syntax:
ROLLBACK;

-- Example:
-- Undo changes made in the current transaction
ROLLBACK;

SQL Joins

Joins are used to combine rows from two or more tables.

1. INNER JOIN

SELECT a.name, b.department
FROM employees a
INNER JOIN departments b
ON a.id = b.emp_id;

2. LEFT JOIN

SELECT a.name, b.department
FROM employees a
LEFT JOIN departments b
ON a.id = b.emp_id;

3. RIGHT JOIN

SELECT a.name, b.department
FROM employees a
RIGHT JOIN departments b
ON a.id = b.emp_id;

4. FULL OUTER JOIN

SELECT a.name, b.department
FROM employees a
FULL OUTER JOIN departments b
ON a.id = b.emp_id;

SQL Functions

SQL provides functions to perform calculations and manipulate data.

1. Aggregate Functions

-- Example: Count, Sum, Avg
SELECT COUNT(*) FROM employees;
SELECT SUM(salary) FROM employees;
SELECT AVG(salary) FROM employees;

2. String Functions

-- Example: CONCAT, LENGTH, UPPER
SELECT CONCAT(name, ' works in ', department) FROM employees;
SELECT LENGTH(name) FROM employees;
SELECT UPPER(name) FROM employees;

3. Date Functions

-- Example: NOW(), DATE_ADD
SELECT NOW();
SELECT DATE_ADD(NOW(), INTERVAL 7 DAY);
